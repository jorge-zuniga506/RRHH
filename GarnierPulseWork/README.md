# Garnier PulseWork

## Descripción
Herramienta para registrar y analizar el estado emocional de los funcionarios en tiempo real.

## Requerimientos Técnicos
- **Backend**: Node.js, Express, SQL
- **Frontend**: Pantalla de registro diario, dashboard de clima
- **IA**: Análisis de sentimientos, detección de patrones emocionales

## Funcionalidades
- Registro diario del estado emocional
- Dashboard con gráficos de tendencia
- Reportes de clima organizacional
- Indicadores de alerta por patrones negativos

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias con `npm install`.
3. Configurar la base de datos.
4. Ejecutar el servidor con `npm start`.
