# Garnier Performance

## Descripción
Sistema para centralizar y automatizar la gestión del desempeño de los colaboradores.

## Requerimientos Técnicos
- **Backend**: Node.js, Express, SQL
- **Frontend**: Dashboard de metas, módulo de reuniones 1:1
- **IA**: Análisis de metas, generación de reportes

## Funcionalidades
- Registro y seguimiento de metas OKR, KPI y MCI
- Módulo de reuniones 1:1
- Feedback bidireccional
- Reportes de desempeño generados automáticamente

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias con `npm install`.
3. Configurar la base de datos.
4. Ejecutar el servidor con `npm start`.
