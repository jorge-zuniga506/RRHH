# Garnier HR Assistant

## Descripción
Agente conversacional inteligente para resolver dudas frecuentes sobre recursos humanos en tiempo real.

## Requerimientos Técnicos
- **Backend**: Node.js, Express, SQL, RAG (Retrieval-Augmented Generation)
- **Frontend**: Interfaz de chat interactiva, panel de administración
- **IA**: Procesamiento de lenguaje natural (NLP), búsqueda semántica

## Funcionalidades
- Carga de PDFs de políticas internas
- Chat en lenguaje natural 24/7
- Respuestas basadas en documentos cargados
- Indicador de fuente y sección consultada

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias con `npm install`.
3. Configurar la base de datos y cargar los documentos iniciales.
4. Ejecutar el servidor con `npm start`.
